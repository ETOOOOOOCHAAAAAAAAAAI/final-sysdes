# myChat

## Описание

простенький чат с возможностью написать в личку голосовых каналов напистать 
---

### старт

### 1. Установка зависимостей

```sh
npm install
```

### 2. Запуск сервера

В одном терминале:
```sh
node server.js
```

Сервер будет слушать порт 3000.

### 3. Запуск нагрузочного теста (30 клиентов)

В другом терминале:
```sh
npm start
```
